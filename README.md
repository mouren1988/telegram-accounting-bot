## 🌟 核心功能特性 (Features)

* ⚡ **极速响应与群发广播**
    * 操作指令 0.1 秒极速响应，告别卡顿。
    * 自动生成清爽账单，后台同步生成详细**网页账单报告**供随时查阅。
    * 按需自定义群发广播，支持图文消息，方便团队通知。
* 📈 **实时汇率与灵活配置**
    * 默认对接欧易 OKX 第三档汇率，**每 60 秒全自动更新**，保障数据绝对实时准确。
    * 支持全局费率/汇率设定，**更支持为群内不同成员（配置）独立设置专属汇率与费率**。
* 🛡️ **群管辅助与安全验证 (Security)**
    * 群内成员修改资料自动提醒，防范仿冒诈骗。
    * 支持 **群组 USDT 地址自动验证**。
    * 私聊机器人发送 TRON (波场) 地址即可秒查详细交易流水记录。
    * 内置智能计算器功能。
* 🕒 **全自动日切管理**
    * 默认每日 06:00 自动进行日切结算（超管可自定义时间）。
    * 账单排版清爽，默认仅展示最近五笔入款和下发记录，保护隐私且不刷屏。

## 📖 核心操作指令速览

本机器人采用自然语言交互，无需复杂的代码基础，会打字就能记账。

### 1. 极简记账
| 指令示例 | 功能说明 |
| :--- | :--- |
| `+1000` | 记录入款 (单位：人民币) |
| `-500` | 修正入款 (单位：人民币) |
| `入款500` / `入账500` | 记录入款 (单位：人民币) |
| `下发100` | 记录下发金额 (单位：USDT) |
| `撤销入款` / `撤销下发` | 撤销最后一笔记录 |

### 2. 汇率与高阶配置
| 指令示例 | 功能说明 |
| :--- | :--- |
| `设置费率 10` | 设置全局手续费率 10% |
| `设置实时汇率` | 开启 60 秒自动更新汇率 |
| `z0` | 查看实时汇率（欧易网） |

💡 **多配置示例：**
* 发送 `设置张三费率5`
* 张三发送 `+10000` 时，系统将自动匹配其专属的独立费率进行计算。

### 3. 权限与系统管理 (超管)
* **权限控制：** `设置所有人` (全员可操作), `设置操作人 @用户名`, `显示操作人`。
* **快捷管理：** 直接回复用户消息发送“设置操作人”或“删除操作人”。
* **系统维护：** `设置日切 00:00`, `删除账单` (清空当日数据), `初始化` (重启服务，不丢数据)。
